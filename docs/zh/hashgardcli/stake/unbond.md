# hashgardcli stake unbond

## 介绍

从一个验证人解绑委托

## 用法

```
hashgardcli stake unbond [validator-addr] [amount] [flags]
```

打印帮助信息

```
hashgardcli stake unbond --help
```

## 示例

```shell
hashgardcli stake unbond \
gardvaloper1m3m4l6g5774qe5jj8cwlyasue22yh32jmhrxfx \
5000gard \
--from=hashgard --chain-id=hashgard
```
